# Item Locations

## General Notes

---

This is a reference guide to the new and/or swapped locations of important items within Sacred Gold & Storm Silver. Some key items are also listed.

This particular document explores two main types of items:

- TMs, of which there have been many swaps
- Evolution items, which have all been shifted to be available prior to the initial Elite Four.
