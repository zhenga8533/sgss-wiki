## :material-information: Effect

!!! info "Description"

    Inflicts regular damage. Has a 10% chance to paralyze the target and a separate 10% chance to make the target flinch.

!!! tip "Quick Summary"

    Has a 10% chance to paralyze the target and a 10% chance to make the target flinch.

## :material-book-open: In-Game Description

!!! quote "Heart Gold & Soul Silver"

    The user bites with electrified fangs. It may also make the foe flinch or become paralyzed.
