??? note "Eevee ROM Changes"

	**Evolution Chain Changes:**

	- `eevee > espeon: level-up (happiness, day)` → `eevee > espeon: use-item (aubade-orb)`

	- `eevee > umbreon: level-up (happiness, night)` → `eevee > umbreon: use-item (serenade-orb)`

	- `eevee > leafeon: level-up (at eterna-forest)` → `eevee > leafeon: use-item (woodland-ore)`

	- `eevee > glaceon: level-up (at sinnoh-route-217)` → `eevee > glaceon: use-item (chilled-ore)`

	**Level-up Moves Changes:**

	- `11 moves` → `13 moves`

	**TM/HM Compatibility Changes:**

	- `21 moves` → `22 moves (added: grass-knot)`
