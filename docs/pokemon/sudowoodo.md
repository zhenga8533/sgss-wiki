# #185 Sudowoodo (Imitation Pokémon)

| Official Artwork | Shiny Artwork |

**Sacred Gold:** Although it always pretends to be a tree, its composition appears more similar to rock than to vegetation.

**Storm Silver:** It disguises itself as a tree to avoid attack. It hates water, so it will disappear if it starts raining.
